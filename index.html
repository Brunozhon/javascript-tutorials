1. Make a Button

This is one of the simplest (and most boring) tasks you have to accomplish.

And here's all of the code you'll have to write.

<button onclick="alert('You clicked me!')">Click me!</button>

2. JavaScript Functions

At least your HTML can be a bit tidier, and it helps with larger programs.

<button onclick="myFunction()">Click me!</button>
...
<script>
  function myFunction() {
    alert("You clicked me!")
  }
</script>

3. Variables

Variables store values, such as "Hello, world!", "You clicked me!", and "Oh, no!".

To create a variable in JavaScript, you just use var, followed by the variable name, followed by the equals sign (which is the = sign in case you didn't know), and finally followed by the value. Confused? Don't worry! It's simply var name = value.

Try creating your own variable name and value below!

var  = ""

var joke = "What thing cracks up the easiest?"
var answer = "An egg."

4. DOM (Document Object Model)

Warning: This section is kind of long.

But before we use variables, let's take a look at the DOM (Document Object Model). This is a tree containing all of the elements in an HTML document, and the document itself.

TREE

HTML
|- HEAD
|  |- TITLE
|  |  |- "Hello, world!"
|- BODY
|  |- H1
|  |  |- "Hello, world!"

ACTUAL DOCUMENT

<html>
  <head>
    <title>Hello, world!</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

Visual 1

If we gave the h1 element an identifier (using the id attribute), the tree and actual document would look like this:

TREE

HTML
|- HEAD
|  |- TITLE
|  |  |- "Hello, world!"
|- BODY
|  |- H1 id="my-heading"
|  |  |- "Hello, world!"

ACTUAL DOCUMENT

<html>
  <head>
    <title>Hello, world!</title>
  </head>
  <body>
    <h1 id="my-heading">Hello, world!</h1>
  </body>
</html>

We then could use document.getElementById("my-heading") to get the element. It's tree will look like this:

H1 id="my-heading"
|- "Hello, world!"

Visuals 2 and 3

If we want the body of the document, you can just use document.body.

After you get the element, you can use the innerHTML property to change the HTML.

<p id="change-html">This is a paragraph.</p>
<button onclick="changeHTML()">Change HTML</button>
...
<script>
  function changeHTML() {
    document.getElementById("change-html").innerHTML = "Now this is an altered paragraph."
  }
</script>

Now, since this was a very long lesson, let's end it with a prank.

And this prank uses the document.write function. Historically, the only way to add HTML to a page was to use the document.write function. Because of that, the HTML parser has to stop just in case the JavaScript interpreter hit this function. When it did, the HTML parser just added some HTML to the bottom of the page. And because you can execute this function at any time, some people decided that if this function was called after the page finished loading, with a BOOM! all of your work will be gone. Except for the message in the function. So now comes the prank:

<button onclick="doNotPressButton()">DO NOT PRESS</button>
<script>
  function doNotPressButton() {
    document.write("Don't say you weren't warned.")
  }
</script>
